About

A lifelong personal and professional interest in technology and passion for learning has lead me to build a skillset as a software developer.

I enjoy the challenge of solving complex problems and employing a fast iteration cycle to spin initial designs into fully function systems and extending exsisting systems based on user feedback.

I have extensive experience in the real estate domain having worked professionally as a State Certified Real Estate Appraiser and as a Real Estate Broker.

As a well-respected real estate appraiser I've worked on highly technical and complex assignments often involving a scarcity of data which required thoughtful and creative solutions. I created a sophisticated proprietary models for analyzing housing trends and for developing market support for adjustments to comparable sales countless hours of time per year. Statistical analysis included using regression models, grouped data analysis on large data sets, and custom quantitative measurements.

As a developer I have worked on several projects including Care Compass which is a proof-of-concept application designed to assist in placing hospital patients in outpatient care facilities. Care Compass is a pure JavaScript application built with an Object Oriented design and includes Google Maps integration and seed data via a public API.

I also worked on a live project for Prosper IT Consulting involving their Learning Management System (LMS). This project was an ASP.NET MVC application. I was responsible for the implementing needed Authentication and Authorization features using the ASP.NET Identity system. This required the creating or user roles including an administrator and a number of new views to support the new, authorized roles. Additionally, I was tasked with implemented UI/UX features to facilitate user navigation of the application including the addition of a progress bar to provide visual feedback.
