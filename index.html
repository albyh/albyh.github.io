About

My initial exposure to computer development was in 1985 on a "portable" Osborne computer using dBase II running on the CP/M operating system. I used a dual floppy-drive setup to create a database for a trade school. That evolved into creating full-featured, custom dBase/Clipper apps for several trade schools and POS software for salons. Later in the mid-1990s as Windows 95 began to get traction I created a successor application in Clipper's successor, CA Visual Objects.

I enjoy the challenge of solving complex problems and employing a fast iteration cycle to spin initial designs into fully function systems.

Over the years I've used Autohotkey extensively to automate common tasks.

As a State Certified real estate appraiser I've created complex spreadsheets for analysis of housing trends and for developing market support for adjustments to comparable sales using analysis of MLS data and statistical analysis using regression analysis, paired sale analysis, and qualatative measurements.
